# P02 - 디버그 출력 + IMU 읽기

## 문제 설명
P01 코드에 시리얼 출력과 IMU를 추가합니다. 시리얼 출력과 센서 읽기를 서로 다른 주기로 돌려 봅니다.

## 요구사항
- P01 기능을 그대로 유지합니다.
- (P01과 동일) 버튼을 계속 누르고 있으면 3초마다 `SAFE <-> ARMED`가 다시 바뀌어야 합니다.
- `Adafruit_MPU6050` 초기화에 성공해야 동작합니다.
- IMU를 50ms 주기로 읽고 `ax, ay, az`를 업데이트합니다.
- 디버그를 100ms 주기로 출력합니다.
- 디버그 형식: `[DBG] t=1234 state=SAFE btn=1 ax=... ay=... az=...`
- 센서 초기화 실패 시 `[ERR] MPU6050 init failed` 출력 후 정지합니다.

## 구현 방향
template.ino의 `TODO A -> TODO C` 순서로 진행하세요.
1. P01 상태/버튼 코드를 먼저 가져와 같은 동작을 만듭니다.
2. IMU 초기화와 50ms 센서 읽기를 구현합니다.
3. 100ms 디버그 출력을 구현합니다.
