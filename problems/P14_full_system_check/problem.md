# P14 - 전체 동작 점검

## 문제 설명
P13 코드를 정리해서 전체 동작을 확인하세요.

## 요구사항
- 상태 흐름: `BOOT -> SAFE -> ARMED -> FLIGHT -> RECOVERY`
- LED 상태 표시, 시리얼 디버그, SD 로깅이 모두 동작해야 합니다.
- 전이 로그에 이유가 출력되어야 합니다.
- 1회 이상 전체 상태 흐름을 확인해야 합니다.

## 구현 방향
template.ino의 `TODO 1 -> TODO 2 -> TODO 3 -> TODO 4` 순서로 진행합니다.
1. P13 코드를 기준으로 사용하고, 함수 이름과 전역변수 이름을 정리합니다.
2. 상태 전이 로그가 모두 출력되는지 먼저 확인합니다.
3. LED 패턴이 상태별로 구분되는지 확인합니다.
4. SD 파일이 생성되고 데이터가 누적되는지 확인합니다.
5. 체크리스트를 순서대로 1회 통과하면 완료로 처리합니다.
