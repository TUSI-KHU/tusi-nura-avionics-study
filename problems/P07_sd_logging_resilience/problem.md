# P07 - SD 로그 추가 (실패해도 계속 동작)

## 문제 설명
P06 상태머신에 SD 저장 기능을 추가합니다. SD가 실패해도 비행 상태머신은 계속 돌아가게 만드는 것이 핵심입니다.

## 요구사항
- `setupLogging()`에서 SD 초기화 및 CSV 헤더를 1회 작성합니다.
- `logData()`에서 100ms마다 `millis,state,accMag,relAlt`를 파일 뒤에 추가합니다.
- SD 오류 시 `logEnabled=false`로 전환하고 상태머신은 계속 동작합니다.
- 파일명은 `flight.csv`를 사용합니다.

## 구현 방향
template.ino의 `TODO 1 -> TODO 5` 순서로 진행하세요.
1. P06 코드를 기본으로 복사합니다.
2. setup에서 SD 로깅 초기화를 붙입니다.
3. loop 끝에서 `logData()`를 호출합니다.
4. 파일 열기/쓰기 실패는 경고만 내고 계속 동작하게 만듭니다.
5. SD 실패 시 로그만 끄고 상태머신은 계속 돌게 만듭니다.
